Unity Coding Adventures

I used Unity to work on some of my personal programming projects.

  • Flocking Simulation: A simple program that emulates the flocking behavior of birds and fish (referred to as boids). It is based on the Flocks, Herds, and Schools paper by Craig Reynolds.
  • Procedurally Generated Worlds: I used the Perlin Noise algorithm to create a randomly generated terrain with hills, shores, and bodies of water.
  • Vision and Navigation: I implemented Dijkstra's algorithm for pathfinding in a grid world. I also developed an algorithm to calculate field of vision.
  • Currently working on creating a turn-based strategy game using the Unity Test Runner to create unit and system tests. So the inevitable bugs won't be too hard to find.

NASA Community College Aerospace Scholars

For our capstone project, my team and I worked on making a robot capable of filming a sporting event (sprint/marathon specifically). This project was a prototype that is part of a $600,000 research grant (which you can read more about in the link below).

  • I was responsible for interfacing with the onboard navigation system (GPS, Motor Controls), using python to control the robots speed and direction relevant to the person it was filming.
  • We used the python OpenCV library to detect the runner and keep him/her at a safe distance.
  • We used two motors and the 3D-printed pan-tilt camera system to keep the runner in frame.
  • Some shins were hurt in the making of this robot.

I joined nine other scholars at JSC NASA to partake in a rover building competition. We were tasked with building a model rover (yes, out of Lego) in order to collect rock samples and identify minerals all while managing our budget and our time.

  • I acted as Lead Software Engineer, writing the programs that controled the rover.
